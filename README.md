HR Analytics Dashboard – Project Overview

HR Analytics Dashboard is a data visualization project designed to analyze and monitor key Human Resource metrics that help organizations make data-driven workforce decisions. The dashboard provides insights into employee performance, attrition trends, salary distribution, hiring patterns, and departmental statistics.

Objective

The main goal of this project is to transform raw HR data into meaningful visual insights that assist HR teams and management in improving employee retention, optimizing recruitment strategies, and enhancing overall organizational performance.

Tools & Technologies Used

Power BI / Tableau – Data visualization and dashboard creation

SQL / Excel – Data cleaning and preprocessing

DAX / Calculated Fields – Advanced metrics and KPIs

Key Features

Employee Attrition Analysis

Department-wise Employee Distribution

Salary & Compensation Insights

Gender and Age Diversity Analysis

Job Role & Performance Tracking

Hiring vs Exit Trends

Interactive Filters and Drill-Down Views

Insights Derived

Identification of departments with high attrition rates

Understanding salary impact on employee retention

Performance trends across different job roles

Workforce diversity evaluation

Business Value

This dashboard helps HR managers:

Reduce employee turnover

Improve hiring decisions

Plan workforce strategies

Enhance employee satisfaction and productivity

Dataset

The dataset contains employee details such as age, gender, department, salary, job role, years at company, performance rating, and attrition status. (Can be real or sample dataset.)
